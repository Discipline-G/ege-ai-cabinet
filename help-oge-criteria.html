Критерии ОГЭ по русскому языку: что значит СК, ГК и ФК

Краткая расшифровка для сочинения-рассуждения 13.1, 13.2 или 13.3.

Главное отличие от ЕГЭ

В ОГЭ критерии сочинения называются не К1–К10, а СК1–СК4 для содержания, ГК1–ГК4 для грамотности и ФК1 для фактической точности.

В официальной работе грамотность ГК1–ГК4 и ФК1 оценивается по изложению и сочинению вместе. ПРОВЕРЯЙ даёт учебную оценку отправленного сочинения.

Содержание сочинения: СК1–СК4

СК1Ответ на вопрос 0–1

Нужно прямо ответить на вопрос, сформулированный в теме сочинения.

СК2Примеры 0–3

Нужно привести примеры, подтверждающие рассуждение. Обычно максимум дают за два подходящих примера из прочитанного текста с пояснением их роли.

Если пример не подтверждает мысль или не связан с текстом, он может не засчитаться.

СК3Логичность речи 0–2

Проверяется, связаны ли мысли между собой, нет ли противоречий, провалов и неправильных переходов.

  • 2 — логических ошибок нет;
  • 1 — одна-две логические ошибки;
  • 0 — три ошибки или больше.

СК4Композиция 0–1

Работа должна быть построена понятно: тезис, доказательства/примеры, вывод. Нарушена композиция — 0.

Максимум по СК1–СК4 — 7 баллов. Если сочинение меньше 70 слов, полностью переписывает/пересказывает исходный текст или написано без опоры на прочитанный текст, оно не оценивается по содержанию.

Грамотность и факты: ГК1–ГК4, ФК1

КритерийЧто проверяетсяШкала
ГК1Орфография: правильное написание слов, -тся/-ться, буквы, слитное/раздельное написание.0 ошибок — 3; 1–2 — 2; 3–4 — 1; 5+ — 0
ГК2Пунктуация: запятые, тире, двоеточия, кавычки и другие знаки.0 ошибок — 3; 1–2 — 2; 3–4 — 1; 5+ — 0
ГК3Грамматика: согласование, управление, формы слов, построение предложений.0 ошибок — 3; 1–2 — 2; 3–4 — 1; 5+ — 0
ГК4Речь: точность слов, повторы, сочетаемость, речевая избыточность или недостаточность.0 ошибок — 3; 1–2 — 2; 3–4 — 1; 5+ — 0
ФК1Фактическая точность речи: нет ли ошибок в фактах, авторах, героях, событиях.Нет фактических ошибок — 1; есть хотя бы одна — 0

Максимум по ГК1–ГК4 и ФК1 — 13 баллов. В официальном ОГЭ эти критерии применяются к суммарному объёму изложения и сочинения.

Как читать результат ПРОВЕРЯЙ

ГК1 0/3 значит: сильно просела орфография. СК2 2/3 значит: примеры есть, но не всё доказано или пояснено.

Выделенные фрагменты в тексте — самые уверенные места ошибок. Сначала исправляй их, потом смотри карту следующей работы.
