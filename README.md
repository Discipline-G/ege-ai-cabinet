# ПРОВЕРЯЙ — Telegram Web App кабинеты и юридические страницы

ПРОВЕРЯЙ — информационно-аналитический ИИ-сервис для учебной самопроверки сочинений ОГЭ/ЕГЭ по русскому языку.

Сайт: https://proveryay.ru  
Бот: @ru_repetitor_bot

> Важно: сервис не является образовательной организацией, официальным экспертом ФИПИ, школой, Рособрнадзором или экзаменационной комиссией. Результаты ИИ-проверки носят учебный и ориентировочный характер.

## Релизные ограничения проверки

- ОГЭ: до 350 слов.
- ЕГЭ beta: до 450 слов.
- Фото: до 4 изображений за одну отправку.
- Работа должна быть чистовой, без задания, черновиков, повторов и лишних материалов.

## Что нельзя хранить в репозитории

Не добавлять в GitHub: токены Telegram/OpenRouter/Yandex/Т-Банка, `.env`, `credentials_decrypted.json`, дампы PostgreSQL, бэкапы n8n, реальные сочинения пользователей, скриншоты кабинетов с персональными данными, логи с Telegram ID, username, токенами, webhook path или содержимым работ.
